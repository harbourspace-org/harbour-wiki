два основных проекта:
1. knottra
2. Harbour.Wiki

1. knottra - app который получает как input сырые данные, как output отдает отформатированную лекцию. Все происходит в реальном времени
2. Harbour.Wiki - приложение где хранятся все записи, это смесь обсидиана, где люди могут оставлять свои коментарии, редактировать материалы и тд. Лекции появляются следующим образом: мы через компьюткры из лекций (преустанавливаем какую-то программу или заходим через зум. скорее всего первое) слушаем звук, картинки с камеры, записи лекций и тд и пропускаем это через knottra. Затем юзеры через mcp подключаются к опеределнной лекции и получают весь свежий контекст

К пятнице я планирую показать только захват звука и трансляцию через mcp.
